# MERN Stack Portfolio

A modern, responsive portfolio website built with the MERN stack (MongoDB, Express.js, React, Node.js).

## Features

- Responsive design that works on all devices
- Dynamic typing animation on home page
- Interactive services section with hover effects
- Tabbed about section with experience, education, and skills
- Projects section with dynamic data from MongoDB
- Contact form with email integration
- Smooth scrolling navigation
- Modern gradient and animation effects

## Tech Stack

**Frontend:**
- React.js
- CSS3 (with custom properties and animations)
- Axios for API calls
- React Icons / Phosphor Icons

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- Nodemailer for email functionality
